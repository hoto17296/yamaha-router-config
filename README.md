# YAMAHA Router Config
YAMAHA Router as Code

## Setup
Create `.env` file

```
USER_PASSWORD=...
ADMIN_PASSWORD=...
PPPOE_DESCRIPTION=...
PPPOE_USERNAME=...
PPPOE_PASSWORD=...
WIREGUARD_SERVER_IP6_ADDR=...
WIREGUARD_SERVER_PORT=51820
```

## Usage
1. `python main.py > config.txt`
2. Import the file into your router
